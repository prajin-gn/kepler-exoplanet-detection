# Literature Review

This document summarizes the key academic papers and online resources
reviewed as part of the project. The focus is on understanding exoplanet detection using Kepler data and the feasibility
of machine learning approaches.

## Research Papers

1. Borucki et al. (2010) – Kepler Planet-Detection Mission  
   - Introduced the Kepler mission and transit photometry concept.

2. Jenkins et al. (2010) – Kepler Science Processing Pipeline  
   - Described the Transit Planet Search (TPS) algorithm.

3. Thompson et al. (2018) – Planetary Candidates Observed by Kepler  
   - Provided labeled datasets suitable for ML classification.

4. Shallue & Vanderburg (2018) – Identifying Exoplanets with Deep Learning  
   - Demonstrated CNN-based transit detection.

5. Armstrong et al. (2018) – ML-based Exoplanet Detection  
   - Discussed feature-based ML approaches and challenges.

## Articles / Blogs (Supplementary)

1. KDAG IIT Kharagpur – *Unveiling the Cosmos: Discovering Exoplanets with ML*  
   - High-level overview of ML applications in exoplanet detection.
